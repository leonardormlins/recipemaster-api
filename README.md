# Recipe Master API
---

## Instructions to run

To run Recipe Master, do you need to execute the instructions bellow:

- Start Mongodb test database:

```bash
docker-compose up -d
```

- Install all dependencies:

```bash
yarn install
```
also works with npm.

- Run functional tests:

```bash
yarn test:functional
```
- Run unit tests:

```bash
yarn test:unit
```
